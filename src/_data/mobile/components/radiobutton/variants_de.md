## Zustände
DIe Komponente hat folgende Zustände: {.lead}
*   Checked
*   Unchecked
*   Disabled checked
*   Disabled unchecked
*   Loading

## Varianten
Die Komponente hat folgende Varianten: {.lead}
*   Radiobutton
    *   ohne Icon / mit Icon
*   Radiobutton-Item
    *   listed / boxed
    *   ohne Icon / mit Icon
    *   ohne Subtext / mit Subtext
    *   ohne Button / mit Button

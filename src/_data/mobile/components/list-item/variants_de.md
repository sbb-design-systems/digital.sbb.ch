## Zustände
DIe Komponente hat folgende Zustände: {.lead}
*   Default
*   Pressed
*   Disabled
*   Loading
*   Swipe left (optional)
*   Swipe right (optional)

## Varianten
Die Komponente hat folgende Varianten: {.lead}
*   listed / boxed
*   mit Icon / ohne Icon
*   mit Subtext / ohne Subtext
*   mit Icon rechts / ohne Icon rechts
*   mit Button / ohne Button

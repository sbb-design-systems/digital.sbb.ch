## Que fait ce modèle ?
* Enregistre les requêtes de recherche ou d'autres préférences des utilisateurs.

## Quand faut-il utiliser ce modèle ?
* Pour les chats IA.

## Recommandation
*  Vérifiez l'importance de cette fonctionnalité pour vos utilisateurs et déterminez si l'historique de recherche doit toujours être affiché ou peut être ouvert sur demande.
*  Les requêtes de recherche doivent être interactives afin que vous puissiez les ressaisir ou les supprimer immédiatement.

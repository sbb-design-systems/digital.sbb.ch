## Cosa fa il pattern?
* Consente l'input tramite un campo di input aperto. 

## Quando dovrebbe essere usato il pattern?
* Quando devono essere effettuate molte richieste diverse, quindi ci si aspetta molta flessibilità.



## Raccomandazione
* Il pattern può sopraffare gli utenti, quindi "Mostra prompt di esempio" o "Mostra esempio iniziale" possono essere d'aiuto (vedi "Suggerimenti").

## Que fait ce modèle ?
* Permet la saisie via un champ de saisie ouvert. 

## Quand utiliser ce modèle ?
* Lorsque de nombreuses demandes différentes doivent être effectuées, ce qui nécessite une grande flexibilité.



## Recommandation
* Le modèle peut dérouter les utilisateurs, c'est pourquoi il peut être utile d'afficher un exemple d'invite ou un exemple de démarrage (voir « Suggestions »).

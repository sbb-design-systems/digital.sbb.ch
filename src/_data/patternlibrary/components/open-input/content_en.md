## What does the pattern do?
* Allows input via open input field. 

## When should the pattern be used?
* When many different requests are to be made, so a lot of flexibility is expected.



## Recommendation
* The pattern can overwhelm users, so ‘Show example prompt’ or ‘Show start example’ can help (see ‘Suggestions’).

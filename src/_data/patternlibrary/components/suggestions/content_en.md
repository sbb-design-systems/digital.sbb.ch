## What does the pattern do?
* Helps users get started by showing what AI can do.

## When should the pattern be used?
* At the start screen
* Also possible as a follow-up question 



## Recommendation
*  Adapt examples to the context. 
*  If user data may be used: Use it to make personalised suggestions.

## Que fait le modèle ?
* Aide les utilisateurs à démarrer en montrant ce que l'IA peut faire.

## Quand le modèle doit-il être utilisé ?
* Sur l'écran de démarrage
* Également possible comme question de suivi

## Recommendation
* Adapter les exemples au contexte.
* Si des données utilisateur peuvent être utilisées : Les utiliser pour faire des suggestions personnalisées.

## Que fait ce modèle ?
* Étend les actions ou les invites à plusieurs cellules ou objets.

## Quand utiliser ce modèle ?
* Lorsque des actions doivent être effectuées sur plusieurs objets.
* Couramment utilisé dans les tableaux.

## Recommandation
* Affichez quelques exemples d'entrées avant de remplir toutes les cellules d'une colonne.
* Faites attention à la manière dont l'IA doit écraser les entrées existantes.

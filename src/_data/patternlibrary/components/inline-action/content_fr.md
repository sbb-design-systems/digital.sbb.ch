## Que fait ce modèle ?
* Il permet d'interagir avec une sortie IA.

## Quand utiliser ce modèle ?
* Lorsque l'utilisateur souhaite modifier ou référencer une partie du texte de sortie.

## Recommandation
* Les actions ne s'affichent que lorsque le texte a été sélectionné.
* La nouvelle invite peut être créée soit dans une boîte de dialogue contextuelle, soit dans le champ de saisie de texte général avec une référence au texte sélectionné.
* Dans le premier cas, le texte est complètement écrasé et l'utilisateur doit donc confirmer l'action.

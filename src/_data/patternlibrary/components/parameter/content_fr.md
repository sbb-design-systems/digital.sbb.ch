## Que fait ce modèle ?
* Permet aux utilisateurs de déterminer le poids de différents éléments.

## Quand le modèle doit-il être utilisé ?
* Lorsqu'un élément tel que le ton, le jeton, le style ou la référence doit être défini. 
* Si un rapport d'image ou une taille d'image doit être défini(e).

## Recommendation
* Étroitement lié au filtre. Vérifier lequel est le plus pertinent.

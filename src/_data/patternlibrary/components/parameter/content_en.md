## What does the pattern do?
* Allows users to determine the weight of different elements.

## When should the pattern be used?
* When an element such as tone, token, style,rReference needs to be defined. 
* If an image ratio or image size is to be defined.


## Recommendation
* Closely related to filter. Check which one makes more sense.

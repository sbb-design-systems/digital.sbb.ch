## Que fait le modèle ?
* Permet aux utilisateurs de donner un retour direct sur une sortie.

## Quand le modèle doit-il être utilisé ?
* Pour les sorties GenAI.
* Si vous souhaitez réinjecter le retour dans le modèle d'IA.


## Recommandation
* Un modèle courant est pouce levé/baissé.
* Communiquer (par exemple après que le retour a été donné) ce qui sera fait du retour.

## Was macht das Pattern?
* Holt mehr Informationen von den Nutzer:innen. 

## Wann soll das Pattern eingesetzt werden?
* Wenn der Input nicht eindeutig verständlich ist.



## Empfehlung
*  Das holen von mehr Informationen kann über eine einfache Rückfrage stattfinden. 
*  Oder durch vorgefertigte Antworten, die zur Auswahl gegeben werden. (Wenn möglich zu verwenden, da weniger Aufwand für Nutzer:innen).

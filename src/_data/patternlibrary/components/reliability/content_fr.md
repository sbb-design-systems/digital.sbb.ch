## Que fait ce modèle ?
* Indique le degré de confiance du système dans sa réponse.

## Quand ce modèle doit-il être utilisé ?
* Dans des contextes « à risque élevé », tels que la législation, les tarifs ou les finances.
* Lorsqu'une décision doit être prise sur la base du résultat.

## Recommandation
* Utiliser une forme bien comprise : Cela peut être un pourcentage, une échelle progressive ou verbale (« probable », « incertain »).

## Que fait ce modèle ?
* Fournit un retour en temps réel sur la qualité de rédaction du prompt.

## Quand le modèle doit-il être utilisé ?
* Peut aider (en particulier les débutants) à rédiger de meilleurs prompts.

## Recommendation
* Selon le contexte, la qualité du prompt peut dépendre d'autres dimensions. Par exemple, la longueur, les détails, la spécificité ou la description contextuelle.

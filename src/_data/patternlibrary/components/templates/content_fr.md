## Que fait le modèle ?
* Aide à la saisie si la rédaction d'une invite serait trop complexe.

## Quand le modèle doit-il être utilisé ?
* Pour simplifier les invites complexes pour la saisie.
* Prédéfini par le système si les principaux cas d'usage sont clairs.
* Dans le cas où les utilisateurs souhaitent créer eux-mêmes des modèles.


## Recommandation
* Qualité plutôt que quantité : Il vaut mieux avoir moins de paramètres mais optimisés.
* Communiquer clairement ce que le modèle fait, par exemple en montrant l'invite générée. De cette façon, les utilisateurs peuvent 
toujours l'adapter.

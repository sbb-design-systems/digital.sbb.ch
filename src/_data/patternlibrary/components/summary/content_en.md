## What does the pattern do?
* Gives a summary of the output either automatically or on demand.

## When should the pattern be used?
* For GenAI outputs.
* If the output is very large. 


## Recommendation
* Name the summary accordingly as ‘summary’.
* Do not use these patterns if the output is short.

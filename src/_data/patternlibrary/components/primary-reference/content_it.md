## Cosa fa il pattern?
* Consente di allegare file o link specifici a cui fare riferimento.

## Quando dovrebbe essere usato il pattern?
* Quando gli utenti desiderano fare riferimento a file o pagine specifiche.


## Raccomandazione
* Verificare per il vostro caso specifico quanti file, la loro dimensione e quali formati di file devono essere consentiti.

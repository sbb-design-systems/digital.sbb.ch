## Que fait ce modèle ?
* Indique les sources utilisées et augmente ainsi la confiance dans le résultat.

## Quand ce modèle doit-il être utilisé ?
* S'il est important que la source soit clairement visible et puisse être trouvée rapidement.
* Dans des contextes « à risque élevé », par exemple lorsqu'il s'agit de lois, de tarifs ou de finances.

## Recommendation
* Pour plus d'une source : Référencer la source au bon endroit dans le résultat.
* Lier les sources afin qu'elles puissent être consultées.
* En option, vous pouvez saisir des informations supplémentaires telles que la date de publication ou le numéro de page.

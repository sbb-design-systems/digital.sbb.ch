## What does the pattern do?
* Communicates what happens to users’ data and lets users decide whether their data should be used.

## When should the pattern be used?
* If you plan to use your users’ data.



## Recommendation
* It should be possible to change a decision also later on, e.g. in the settings. 
* Consider where to set opt-in and opt-out as default.

## Que fait ce modèle ?
* Il communique ce qu'il advient des données des utilisateurs et leur permet de décider si leurs données doivent être utilisées.

## Quand utiliser ce modèle ?
* Si vous prévoyez d'utiliser les données de vos utilisateurs.

## Recommandation
* Il doit être possible de modifier une décision ultérieurement, par exemple dans les paramètres. 
* Réfléchissez à l'endroit où définir l'opt-in et l'opt-out par défaut.

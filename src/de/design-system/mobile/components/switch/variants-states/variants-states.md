## Zustände
DIe Komponente hat folgende Zustände:{.lead}
*   Selected
*   Unselected
*   Disabled Selected
*   Disabled Unselected
*   Loading
*   Error (default / retry / reloading / button)

## Varianten
Die Komponente hat folgende Varianten:{.lead}
*   Switch
    *   Selected / Unselected
*   Switch-Item
    *   without icon / with icon
    *   listed / boxed
    *   without subtext / with subtext
    *   without link / with Link

## Zustände
DIe Komponente hat folgende Zustände:{.lead}
*   Default
*   Error
*   Disabled
*   Loading
*   Expanded

## Varianten
Die Komponente hat folgende Varianten:{.lead}
*   listed / boxed
*   mit Icon / ohne Icon
*   mit Label / ohne Label
